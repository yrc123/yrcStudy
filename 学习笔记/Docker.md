## 更换旧容器

1. 获取旧容器ID
   - 使用`docker ps`获取
2. 停止旧容器
   - 使用`docker stop [id]`停止容器
3. 删除旧容器
   - 使用`docker rm [id]`删除容器
   - 也可以使用`docker rm -f [id]`来**停止并暂停**容器
4. 创建新容器
   - 使用`docker run [容器名]`创建容器
